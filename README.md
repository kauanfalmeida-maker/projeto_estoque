# 🚀 Projeto_Estoque

Um sistema de estoque simplificado:

## FUNCIONALIDADES:

Digite 1 para listar os itens do estoque
Digite 2 para solicitar o ID do produto, caso ele exista 
Digite 3 para registra um produto que ainda não esta na lista (id, nome, quantidade, localização) 
Digite 4 para atualizar as quantidades de produtos no estoque 
Digite 5 para sair do estoque 

Sistema funcional de estoque!!
